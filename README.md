# Estudos de Direito

Repositório de estudos interativos para o Curso de Direito.

O objetivo é publicar experimentos em HTML, CSS, JavaScript, Canvas e 3D que facilitem revisão e consulta em desktop e mobile.

## Estrutura

- `index.html`: catálogo geral do GitHub Pages.
- `disciplinas/introducao-ao-estudo-do-direito/`: índice da disciplina.
- `disciplinas/introducao-ao-estudo-do-direito/piramide-de-kelsen/`: experimento 3D da Pirâmide de Kelsen brasileira.
